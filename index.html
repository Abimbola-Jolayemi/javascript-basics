STATICALLY-TYPED PROGRAMMING LANGUAGE

It is a type of programming language that follows type checking during compilation

Examples are: JAVA, C, C++, ForTran

DYNAMICALLY-TYPED PROGRAMMING LANGUAGE

In this type of language, type checking takes place while the program runs

Examples are: Python, Javascript, PHP, Ruby
